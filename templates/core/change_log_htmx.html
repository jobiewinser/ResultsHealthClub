V0.5 - 06 December 2022
  • Remade the system for assigning auto send templates to a campaign, now can assign up to 10 templates per campaign. Added a seperate popup window for adding new templates. Removing the template choices from the campaign configuration page has freed up some space.
  • Added new concept of a campaign type to allow the system to cater to wider spanning gyms (i.e the system can similtaneously be used for group class leads and a PT training course for example).
  • Added Google Sheets style "Who's Working" section with profile pictures at the top of the leads page showing who is currently on the lead board! Very cool!
  • Made an Edit Lead Section. Currently only on the leads page, soon to be on the booking page too.
  • Made the leads columns equal lengths again.
  • Made card dragging a little smoother. Still more to be done though.
  • Added a character limit to each section of the whatsapp template with explanations on why.
  • Added frontend validation on the whatsapp template name to prevent users accidentally submitting an invalid name.
  • Made submitting a new whatsapp template a little more responsive (it tells you what it's doing in the background).
  • Added double clicking leads (or tapping) to make it touch friendly. Mobile now kind of supported.
  • Restyled the analytics page a bit more.
  • Booking table now sorted by booking date... Duh!
  • Added Date booking made in booking history.
  • Added a New Login Page. Image is placeholder.
  • Login username/email is not case sensitive any more.
  • The value of a lead is now determined upon creation, based on the campaign value at that moment in time. This can be edited in the new edit lead popup.
  • Added search to various select boxes (select2 library).
  • Bugs:
    • Fixed double scroll bars on small screen devices.
    • Fixed leads not being claimable in some scenarios.
    • Fixed leads card not wanting to nicely go into the archive/booking boxes.
    • Fixed changing sites on the whatsapp template table not working.
